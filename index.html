Free Open Ghost

FOG rivals its paid competitors. Supporting nearly all operating systems, including Microsoft Windows, Apple OSX, and Linux based, FOG is compatible with nearly all environments.

Work From Anywhere

FOG uses a web portal, allowing you to manage your network from anywhere and from any device. Schedule image deployment, software deployment, or even turn off computers from the comfort of your smartphone.

Computer Management

Install software, run scripts, add printers or even activate Windows straight from the web portal.

Multiple Sites

Have offices accross states, countries, or even across the world? FOG can image and manage all of them. Simply set up a FOG node in each of your building.

Off Site Support

Need to manage computers that aren't in company offices? No problem, computers can be managed whever there is internet.

Support

FOG is built around an active community. If you encounter a problem or just have questions post on the forums and get a response almost immediately. Not only will you have access to others who use FOG but also direct access to the development team who interact with the community daily.

Flexibility

Want to customize FOG to meet your enviornment? That is not only possible but encouraged, because FOG is open source you have complete control over your server. Get in touch with a developer by posting on the forums. Want a new feature but don't know how to program? Not a problem, simply post a feature request and the development team will take a look at it.

Security

When it comes to managing computers, security is of the essence. The next stable release will include a new secuirty system that ensures server & client trust. Both the client and the server will communicate using a secure socket tunnel encrypted using RSA, AES, and certificate pinning.
